Haskell Learning Resources.

Posted on January 15, 2020

This is my list of Haskell Learning Resources, mostly for my own use, although I am sometimes asked for my recommendations, and maybe you will find it useful.

Books

Haskell from First Principles
Probably the best way to learn haskell, very in-depth and with a friendly approach.
http://haskellbook.com/

Learn you a Haskell for great good A great free book to give you your first taste of Haskell. http://learnyouahaskell.com/

Thinking Functionally in Haskell
https://www.amazon.co.uk/Thinking-Functionally-Haskell-Richard-Bird/dp/1107452643

Real World Haskell
http://book.realworldhaskell.org/
This Github repo has info which brings this book up-to-date

Thinking in Types
https://leanpub.com/thinking-with-types

A Type Of Programming
https://atypeofprogramming.com/

Finding Succss and Failure in Haskell
https://joyofhaskell.com/
leanpub / ebook
lulu / real book

Bartosz Milewski: Category Theory for Programmers
Website
PDF ebook

No longer active :’(

They usually pick a topic which was mentioned in the weekly newsletter. https://haskellweekly.news/podcast/

LambdaCast My highly recommended podcast for FP beginners. https://soundcloud.com/lambda-cast

CoRecursive
General FP podcast that regularly features haskell content.
https://corecursive.com/
